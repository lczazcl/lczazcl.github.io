基本类型：Number、String、Boolean

Function

Function 构造函数 创建一个新的Function对象。 在 JavaScript 中, 每个函数实际上都是一个Function对象。

语法：

new Function ([arg1[, arg2[, …argN]],] functionBody)
arg为参数，[]内的参数代表可选(传或不传)，functionBody代表方法体(函数体)。

Function和function的区别

function是一个关键字，等同于if eles var function
Function是一个全局对象，等同于window.Object
